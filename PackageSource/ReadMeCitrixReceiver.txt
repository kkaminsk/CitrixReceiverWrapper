Citrix Reciever MSI Wrapper by Big Hat Group Inc.
This installer is provided as-is to install the latest Citrix Reciever via a Windows Installer Package for deliver via Mobile 
Device Management or manual software installation. This installer requires roughly 250 megabytes of free disk space, Windows 7
or better, administrator rights to the machine and an Internet connection. Supported actions for this wrapper are install and uninstall.
